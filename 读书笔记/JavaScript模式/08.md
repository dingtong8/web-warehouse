## 外观模式

它为对象提供了一个可供选择的接口

```js
var myevent = {

    // ...

    stop: function (e) {
        e.stopPropagation();
        e.preventDefault();
    }

    // ...
}
```

外观模式也非常适合于浏览器脚本处理，据此可将浏览器之间的差异隐藏在外观之后

```js
var myenevt = {

    // ...

    stop: function (e) {

        // 其他
        if (typeof e.preventDefault === "function") {
            e.preventDefault();
        }

        if (typeof e.stopPropagation == "function") {
            e.stopPropagation();
        }

        // IE 浏览器
        if (typeof e.returnValue === "boolean") {
            e.returnValue = false;
        }

        if (typeof e.canceBubble === "boolean") {
            e.canceBubble = true;
        }
    }

    // ...
    
}
```
